App: Local Machine translation (translate)ÔÉÅ

The translate app is one of the apps that provide machine translation functionality in Nextcloud and act as a translation backend for the Nextcloud Assistant app, the text app and other apps making use of the core Translation API. The translate app specifically runs only open source models and does so entirely on-premises. Nextcloud can provide customer support upon request, please talk to your account manager for the possibilities.

The app currently supports the following languages:

  • English (en)

  • German (de)

  • French (fr)

  • Spanish (es)

  • Chinese (zh)

As the models are entirely open source, the quality of translations may not be comparable to commercially available machine translation services.

RequirementsÔÉÅ

  • Minimal Nextcloud version: 26

  • x86 CPU

  • GNU lib C (musl is not supported)

  • This app does not support using GPU for processing and may thus not be performing ideally for long texts

(Note: Nextcloud AIO is currently not supported due to it using musl)

Space usageÔÉÅ

  • ~1GB per language pair

  • ~10GB in total

InstallationÔÉÅ

  1. Make sure the Nextcloud Assistant app is installed

  2. Install the translate app via the ‚ÄúApps‚Äù page in Nextcloud, or by executing

    occ app:enable translate

SetupÔÉÅ

After installing this app you will need to run:

occ translate:download-models

You may also download only specific languages by using the following command:

occ translate:download-models <languages>

For example

occ translate:download-models de en

will download both en->de and de->en.

occ translate:download-models de en es

will download en->de, de->en, en->es, es->en, es->de, de->es

It‚Äôs recommended to only use AI at the beginning of a creation process and not at the end, so that outputs of AI serve as a draft for example and not as final product. Always check the output of language models before using it.

  • Make sure to test the translation model you are using it for whether it meets the use-case‚Äôs quality requirements
